1. Experimental Setting

-The “Training Set 1” dataset is used for training model

- The “Test Set 1” dataset is used for testing model

2. Evaluation metrics

AP (AveragePrecision) is used as a measure to evaluate the model. AP is computed by calculating the average interpolation of the precision values over the recall values in [0,1]. We use the interpolation method at 101 recall points. which includes: APIoU=0.5 is the AP value at the threshold of IoU is 0.5, APIoU=0.75 is the AP value at the threshold of IoU is 0.75, and AP is the average AP value for the threshold of IoU from 0.5 to 0.95 with each increment of 0.05.
